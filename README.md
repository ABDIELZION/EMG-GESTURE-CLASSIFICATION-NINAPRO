# EMG Gesture Classification (NinaPro DB4)

This project demonstrates a Machine Learning pipeline to classify hand movements using 12-channel EMG sensor data.

## 📊 Results
- **Algorithm:** Random Forest Classifier
- **Accuracy:** 70% (Proof of Concept)
- **Features:** Mean Absolute Value (MAV)
- **Window Size:** 100ms (200 samples @ 2kHz)
